AI 求职打招呼助手
==================

Chrome 扩展（Manifest V3）：在招聘网站抓取岗位 JD，结合简历亮点用 AI 生成打招呼语并一键复制到剪贴板。
“每一份岗位都值得被认真对待，每一位求职者都不该被系统模版埋没。”

我做这个插件的初心，是因为觉得系统自动生成的打招呼语缺乏灵魂。我希望通过 AI 的能力，实现**效率与个性化的完美兼顾**。它不仅能根据不同岗位自动调整话术，更能融入求职者真实的人格特质，以此提高 HR 的回复率，让沟通更有温度，让效率真正服务于人。

功能
----
· 上传简历：支持 .txt、.pdf，自动提炼简历精华并缓存，下次生成时优先使用缓存；整页仅一处显示「未选择任何文件」或当前文件名。
· 一键生成：在 Boss 直聘、拉勾、LinkedIn 等职位详情页点击「一键生成打招呼并复制」，自动完成：保存当前选项 → 抓取当前页 → 清洗 JD → 生成打招呼话术 → 复制到剪贴板；弹窗内有一条实时进度提示，失败时会在弹窗中显示错误信息。
· 话术风格：诚恳型、活泼型、数据型三种风格可选。
· AI 接口：支持 OpenAI 兼容与 Gemini 兼容两种接口，在弹窗中选择即可；API 端点已在扩展内配置，使用前只需填写 API Key。


安装
----
1. 在 Chrome 中打开 chrome://extensions/ ，开启「开发者模式」。
2. 点击「加载已解压的扩展程序」，选择本项目的根目录（包含 manifest.json 的文件夹）。
3. 点击扩展图标，在弹窗中填写 API Key，选择 AI 接口（OpenAI 兼容 / Gemini 兼容）。


使用步骤
--------
1. 点击扩展图标，在弹窗中填写 API Key，选择 AI 接口（OpenAI 兼容 / Gemini 兼容）。
2. 点击「选择文件」上传简历（.txt 或 .pdf），右侧会显示当前文件名；上传后会自动提炼并保存简历精华。
3. 选择话术风格（诚恳型 / 活泼型 / 数据型）。
4. 在招聘网站打开职位详情页，再点击扩展图标，点击「一键生成打招呼并复制」。
5. 保持弹窗打开，等待弹窗内进度提示（正在读取岗位页 → 正在生成打招呼）；完成后页面上会弹出「打招呼已生成，立刻开聊吧！」，话术已复制，在聊天框粘贴即可。若报错会在弹窗中显示原因。


项目结构
--------
greeting/
  manifest.json        扩展配置
  popup.html           弹窗页面
  popup.js             弹窗逻辑（上传、生成流程）
  style.css            弹窗样式
  background.js        后台：调用 AI（清洗 JD、提炼简历、生成话术）
  content.js           注入页：抓取页面文字、复制到剪贴板、Toast
  libs/
    pdf.min.js         PDF.js 主库（解析 PDF 简历）
    pdf.worker.min.js  PDF.js Worker
  README.txt           本说明（中文）
  README-en.txt        说明（英文）


数据与隐私
----------
· API Key、AI 接口、话术风格等选项保存在浏览器本地（chrome.storage.local），仅在与对应 AI 接口通信时使用。
· 简历原文、简历精华、上次清洗的 JD 会缓存在本地，用于下次生成时加快流程。
· 支持的招聘站包括 Boss 直聘（zhipin.com）、拉勾（lagou.com）、LinkedIn 等；其他站点会整页抓取后由 AI 清洗出 JD。
